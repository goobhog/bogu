Bogu uses SBCL http://www.sbcl.org/ and Csound https://csound.com/

Bogu is a symbolic language for composing music on the computer.
Each note of each octave has its own symbol; 
notes and symbols can be combined to form more complex symbols,
which can then be used to write higher and higher orders of interesting music.

Bogu is free and open-source, feel free use, share and build upon.
